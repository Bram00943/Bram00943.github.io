About Weatherapp

This was a very quick application where I've used a Excel file and Unity. You can give in the Excel file various value's and then the icons in Unity change depending on the value's you gave in the Excel file. So in Antartica it's obviously very cold so the snowflake icon is bigger then in other country, but in NY it's not that cold so the snowflake is a little smaller.

My contribution

I wouldn't say I'm not proud of this project, but I admit that I have made better projects. Only because this one was just a quick project that I made to look if you can work with an excel file and adjust values from within the Excel file and change things inside of Unity.

Conclusion

In short I am now aware then I cannot make just small projects because then they will be boring to 'play' or execute. So whenever I want to create a new project I need just some time to develop the project and think of interesting things I can add to make it more ejoyable to play or execute.

Over Weatherapp

Dit was een zeer snelle applicatie waarbij ik een Excel-bestand en Unity heb gebruikt. Je kunt in het Excel bestand verschillende waardes opgeven en dan veranderen de iconen in Unity afhankelijk van de waarden die je gaf in het Excel-bestand. Dus in Antartica is het is het duidelijk erg koud dus het sneeuwvlok icoon is groter dan in andere landen, maar in NY is het niet zo koud dus is de sneeuwvlok een beetje kleiner.

Mijn bijdrage

Ik zou niet zeggen dat ik niet trots ben op dit project, maar ik geef toe dat ik betere projecten heb gemaakt. Alleen omdat dit gewoon een snel project was dat ik maakte om te kijken of je kunt werken met een Excel-bestand en waarden kunt aanpassen vanuit het Excel bestand en dingen veranderen binnen Unity.

Conclusie

Kortom gezegd, ik ben me er nu van bewust dat ik niet alleen maar kleine projecten kan maken omdat dan zullen ze saai zijn om te 'spelen' of uit te voeren. Dus wanneer ik een een nieuw project wil maken heb ik wat tijd nodig om het project te ontwikkelen en interessante dingen te bedenken die ik kan toevoegen om het leuker te maken om te spelen of uit te voeren.
