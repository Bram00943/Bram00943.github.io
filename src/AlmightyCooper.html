About Almighty Cooper

This is a console based game, where I used a sort of dice based gameplay where you get to 'roll' a dice and you fight a monster which is called Almighty Cooper, and if you roll a certain amount from 1-6 it's multiply's by a amount. And from there on it damages Cooper and you can select the amount of player counting up to 4 and you can give them names and there are here and there cool features.

My contribution

Almight Cooper was a pretty hard console game. Because it was a lot of code (in my opinion) for just a few little things. But I asked help pretty often, which was, when I look back not the best option. Because I just instantly asked for help instead for looking for an solution. And a few things were pretty hard to understand because I was still quiet new to making a console game, especially with the dice aspect.

Conclusion

I sort of liked making it but it's quiet sad that it's a console game instead of an actual game powered by I don't know Unity, Unreal Engine. But I still think it looks cool but I cannot do anything else with it except just playing it on my Visual Studio and after a very short period of time it gets a little boring to be honest. But it worked out pretty well because it's a game and it works.




Nederlands
Over Almighty Cooper

Dit is een console-gebaseerd spel, waar ik een soort dobbelsteen-gebaseerde gameplay heb gebruikt waar je een dobbelsteen gooit en je vecht tegen een monster genaamd Almighty Cooper, en als je een bepaald aantal ogen van 1-6 gooit wordt het vermenigvuldigd met een aantal. En vanaf daar kan je Cooper aanvallen, ook kan je het aantal van de speler kiezen tot een aantal van 4 spelers en je kunt ze namen geven en er zijn nog meer gave functies.

Mijn bijdrage

Almight Cooper was een behoorlijk moeilijk consolegame om te maken. Omdat het een heleboel code was (naar mijn mening) voor maar een paar kleine dingen. Maar ik vroeg vrij vaak hulp, wat, als ik terugkijk niet de beste optie was. Omdat ik gewoon direct om hulp vroeg in plaats van te zoeken naar een oplossing. En een paar dingen waren vrij moeilijk te begrijpen omdat ik nog vrij nieuw was met het maken van een console spel, vooral met het dobbelsteen aspect.

Conclusie

Ik vond het leuk om te maken, maar het is jammer dat het een console spel is in plaats van een echt spel aangedreven door, ik weet het niet, Unity of Unreal Engine. Maar ik denk nog steeds dat het er goed uitziet qua kleuren enzo, maar ik kan er niets anders mee doen dan het gewoon spelen op mijn Visual Studio en na een zeer korte periode van tijd wordt het een beetje saai om te spelen. Maar het werkte vrij goed omdat het een spel is en met een game kan er niet zoveel mis gaan.
