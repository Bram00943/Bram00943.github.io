About YT remake

Here I created an remake of the video sharing platform YouTube. It has just the basic functions, you can click on the video, have a little nav bar, has a modal when you enter the page. This page was made in HTML with Bootstrap, and a little styling with CSS.

My contribution

After I finished the layout that I had created for the website, I decided to try to add some cool features like YouTube has. But it turned out that was way more diffucult then I thought it be. I still gave it a few attempts on my own and try to make a search bar when if you type for a certain song/video it shows only that song that you searched for. Though I was very close it didn't work perfectly. That was very discouraging, I wouldn't say that was the reason I quit working on it, but I kinda lost hope because it was hard. And the solutions that I did found didn't really work on my website.

Conclusion

Now I've learned that it's pretty hard to make a search function that makes sure you only get the video you have searced for, and even with checkboxes it was still hard, because then I tried putting the videos in different classes which they represent. Like music, footbal, gaming, so when you check that class those videos showed up, but that also didn't work in the end.
