About CSS box

This is made in HTML with Css and here I set a min width where the variables are the ones for a tablet and a desktop, so when you switch to different devices the box's colums change accordingly.

conclusion

This went pretty easy because I figured it out pretty quickly how to change the boxes with different device resolutions, this I wanted to test because I had some issues with scaling boxes in previous websites so I went on and made a very quick website to test it.

Over Css box

Dit is gemaakt in HTML met Css en hier stel ik een minimale breedte in waar de variabelen zijn die voor een tablet en een desktop, dus wanneer u overschakelt naar verschillende apparaten, de kolommen van de box veranderen dienovereenkomstig.

Conclusie

Dit ging vrij gemakkelijk omdat ik er vrij snel achter kwam hoe ik de dozen met verschillende apparaatresoluties, dit wilde ik testen omdat ik problemen had met schalen vakjes op eerdere websites, dus ging ik verder en maakte een zeer snelle website om het te testen.
