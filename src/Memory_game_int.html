About Memory Game

This is a project I have been working on when I had my internship at @LiveWall. This project was made with JS React and scss. This minigame is one of many they use, they needed some new minigames so I was asked to make a memory game. Now for starters it is not the hardest thing to code but when you have very little to none experience with react, it become quite the challenge.

My contribution

Since I was so used to work with basic html css and js, and then suddenly working with a brand new framework of which I've never heard of was a big challenge. But of course there were people working in company who obviously had way more experience with JS React that I did, so whenever I couldn't figure something out I could ask for help and they would help me out. The mobile phone page started out with basically just a preset of the basic layout of their company and nothing else. So I had to google and ask a few times how to start and they told me I should break it up into little pieces, that way I could develop the application step by step.
