About Malaysia Inhabitants

This is a project where I've worked on with someone else. For this project we needed to get data from a webservice, where we used this postman API to get the Inhabitants of malaysia. Our innitial goal was to get the amount of people vaccinations to show but while developing we accidentally collected and showed the data of the total Inhabitants of malaysia. We've done this per state so you could click on the state and it would show the amount of Inhabitants of that state. This project had a deadline, which meant we had to get our priorities straight. Since our innitial goal was to display the amount of vaccinated people in a state and instead we got the amount of Inhabitants per state, we kind of missed our goal.

My contribution

I have mainly been focussing on the front-end of this application since that is where my interest lies most. The map itself is essentially just an SVG which I've modified so that if you hover a certain state the color of that state in the svg changes. First the whole svg changed colors but after some testing and developing I got it the way it is now. I essentially styled everything on the page that you see in the video, added some transitions here and there. When we were near the end of the project I tried some developing in the backend aswell but the time window was too small so that's why you'll see in the video data missing since that is what we could not finish within the deadline of the project.
