About Todo App

Well, I've made a todo app in the past. But that one wasn't really working well, and had some bugs I wasn't able to fix back then. So I decided to make a brand new one. This one works almost perfectly and had no bugs, though there are some small functions that are not working because I didnt add them, it still works fine!

Conclusion

I've learned that when doing a little bit of research you can do a lot. because like other websites you start from scratch and just build whatever comes to mind. And I didn't really have an idea to what color pallet I should use, but in my first portfolio website I've created I used a color pallet which I created myself so I used that one. And I think it is also pretty user friendly if my todo app were public.




Nederlands
Over Todo App

Nou, ik heb in het verleden een todo-app gemaakt. Maar die werkte niet echt goed, en had een aantal bugs die ik toen niet kon oplossen. Dus besloot ik een nieuwe te maken. Deze werkt bijna perfect en had geen bugs, hoewel er enkele kleine functies zijn die dat niet zijn werkt omdat ik ze niet heb toegevoegd, het werkt nog steeds prima!

Conclusie

Ik heb geleerd dat als je een beetje onderzoek doet, je veel kunt doen. omdat je net als andere websites helemaal opnieuw begint en gewoon wat dan ook bouwt komt in je op. En ik had niet echt een idee tot welk kleurenpalet ik zou moeten gebruiken, maar in mijn eerste portfoliowebsite die ik heb gemaakt, heb ik een kleur gebruikt pallet die ik zelf heb gemaakt, dus die heb ik gebruikt. En ik denk dat dat ook zo is behoorlijk gebruiksvriendelijk als mijn todo-app openbaar was.
