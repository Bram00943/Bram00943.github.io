About Main Fielders

This is also an quick website made up for small Javascript exercises. So I did a little css styling here and there, and this is just simple Javascript I did becuase I still want to improve my Javascript knowledge. So I started with input fields where you can input anything and it replaces it with the text that says: "Change meee", and some other quick examples of Javascript. The last one was the most difficult, it's called Fibonacci. That is when 1,1 = 2, and then it uses the last number and adds the previous number to it. so when it's 1,1 = 2 then the next number is gonna be 3, because 1+2 = 3. And then the next number would be 5, because 3 was the last number and the number before that was 2. And that just goes on and on.

Conclusion

Like said previously, the Fibonacci was the hardest to make because I'm still learning Javascript. And the calculation of adding the numbers into an array and adding the last number to the other was so difficult. But with some searching on the internet I was able to figure it out.

Over Main Fielders

Dit is ook een snelle website die is gemaakt voor kleine Javascript-oefeningen. Dus ik deed een beetje CSS-styling hier en daar, en dit is gewoon simpel Javascript dat ik deed omdat ik mijn Javascript-kennis nog steeds wil verbeteren. Dus ik begon met invoervelden waar u alles kunt invoeren en het vervangt het door de tekst die zegt: "Change meee", en enkele andere snelle voorbeelden van Javascript. De laatste was het moeilijkst, het heet Fibonacci. Dat is wanneer 1,1 = 2, en dan het gebruikt de computer het laatste nummer en voegt het vorige nummer eraan toe. Dus als 1,1 = 2 is dan wordt het volgende getal 3, want 1+2 = 3. En dan het volgende getal zou 5 zijn, want 3 was het laatste getal en het getal daarvoor was 2. En dat gaat als maar door.

conclusie

Zoals eerder gezegd was de Fibonacci het moeilijkst om te maken want ik ben nog steeds Javascript leren. En de berekening van het optellen van de getallen in een array en het laatste getal bij het andere optellen was zo moeilijk. Maar met wat zoeken op de internet Ik heb het kunnen achterhalen.
