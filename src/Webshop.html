About Shoppers

This is a small website where you can buy items. This is made in HTML with a little css stylings so that it's not just a plain website, and Javascript. For this website I didn't really care too much about the styling, because I was mainly focused on the Javascript part. So you can buy a certain amount of the items, and when it reaches 0 it goes out of stock obviously. And it stores the amount of items in variables and add them later in the end together with the other purchased items. Then the total price shows but without the VAT, and also the VAT is being calculated based on the price of the purchased items total. Then the absolute total is shown.

Conclusion

I wouldn't say this was 'hard', but it was quite challenging in the beginning. Because a lot was based on Javascript and I'm still learning the language, but in the end I got it all working. The hardest part for me was calculating the total and then getting the VAT of the total. But I got it in the end, and when I look back at my code that I've written, it could maybe be more optimized. Because I've used more valuables over and over but added a number behind it, but I couldn't think of a way to write that more efficiently.

Over Shoppers

Dit is een kleine website waar je artikelen kunt kopen. Dit is gemaakt in HTML met een beetje css-styling zodat het niet een saaie website is, en Javascript. Voor deze website heb ik gaf niet zo veel om de styling, want ik was vooral gefocust op de Javascript-gedeelte. U kunt dus een bepaald aantal items kopen en wanneer het 0 . bereikt het is duidelijk uitverkocht. En het slaat het aantal items op in variabelen en voeg ze later op het einde samen met de andere gekochte items toe. dan het totaal prijs wordt weergegeven, maar zonder de btw, en ook de btw wordt berekend op basis van de totale prijs van de gekochte artikelen. Vervolgens wordt het absolute totaal weergegeven.

Conclusie

Ik zou niet zeggen dat dit 'moeilijk' was, maar in het begin was het behoorlijk uitdagend. Want veel was gebaseerd op Javascript en ik ben de taal nog aan het leren, maar uiteindelijk heb ik het allemaal werkend gekregen. Het moeilijkste voor mij was het berekenen van het totaal en dan de btw van het totaal krijgen. Maar ik heb het uiteindelijk gekregen, en als ik terugkijk op mijn code die ik heb geschreven, kan het misschien meer worden geoptimaliseerd. Omdat ik heb gebruikt steeds meer waardevolle spullen maar er een nummer achter toegevoegd, maar ik kon geen manier bedenken om dat efficiënter te schrijven.
